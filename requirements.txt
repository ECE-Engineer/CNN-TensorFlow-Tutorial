Using Virtual ENV
===============================================================================
Linux
If on the nuc and want to utilize the cores on the sever use X-11 forwarding:
ssh -X <server-name>

sudo apt-get install python
sudo apt install python-pip
python -m pip install --user virtualenv
python -m virtualenv p_2
source p_2/bin/activate.csh
pip install --upgrade tensorflow
pip install --upgrade scikit-plot
python mnist_v2.py
deactivate

sudo apt-get install python3
sudo apt install python3-pip
python3 -m pip install --user virtualenv
python3 -m virtualenv p_3
source p_3/bin/activate.csh
pip3 install --upgrade tensorflow
pip3 install --upgrade scikit-plot
python3 mnist_v2.py
deactivate
